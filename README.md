    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a name="user-content-bleshield_shutter" class="anchor" href="#bleshield_shutter"><span class="octicon octicon-link"></span></a>BLEShield_shutter</h1>

<p>The BLE shutter control using BLE shield.
This project is based on the Nordic BLE library and the BLE shield V2.0 from RedBearLab.
You can get it from the following link.
<a href="https://github.com/NordicSemiconductor/ble-sdk-arduino">https://github.com/NordicSemiconductor/ble-sdk-arduino</a>
You can download it and import it into your Arduino environment.</p>

<p>The project has been tested on iphone5 with iOS 7.1.</p>

<p>Note that this function uses the BLE profile called HID over GATT, and it is not supported until iOS7.
Also, iDevices support BLE after iphone4S.</p>

<p>About the hardware connection, you can visit the wiki for detail.</p></article>
